Portfolio Projects

  • Emotion Recognition System: A real-time facial emotion detection system using Convolutional Neural Networks (CNN) and the FER2013 dataset. Implemented image augmentation and live webcam streaming to classify emotions accurately.
  • OCR Scanner: A Python application leveraging OpenCV and Tesseract to extract editable text from images and export it to .doc or .txt formats. Great for digitizing printed materials.
  • EduSentral Entrance System: An Android app using Java and SQLite that allows students to take secure online entrance exams on their smartphones. Built with a robust backend and clean mobile UI.
  • Store Sales Forecasting: A predictive analytics model using historical sales data from Kaggle. Implemented regression algorithms and time series analysis to estimate future store performance.
  • Nepali Plagiarism Detector: A unique NLP-based tool using rule-based stemming and cosine similarity to detect plagiarism in Devanagari script documents. Useful for academic verification.

Blog / Reflection

Why I Chose QA and Automation

Software quality assurance may seem like a background process, but it’s one of the most critical components in tech. I was drawn to QA because of the meticulous planning it requires and the challenge of building systems that catch what others might miss. Automation has elevated QA to a whole new level—allowing us to test faster, smarter, and more reliably.

Machine Learning in Everyday Life

From product recommendations to voice assistants and fraud detection, machine learning touches our lives in countless invisible ways. One of my goals is to demystify ML for everyday users—making these technologies more accessible and impactful. I'm currently working on integrating ML models into practical tools that help in education and e-commerce.
